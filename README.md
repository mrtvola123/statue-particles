# Liberty Particles

Interactive Three.js particle reconstruction of the Statue of Liberty. The app samples a real GLB mesh into a 70,000-particle point cloud; click once to disintegrate it, click again to rebuild it.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Verify

```bash
npm test
npm run build
npm run test:e2e
```
